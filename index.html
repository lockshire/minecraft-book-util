Minecraft Book Command Utility

Enter your book info below and click da button.

About

This utility generates a Minecraft /give command to generate a book in-game via a command block.

It's primarily intended to generate longer, multi-page books without having to worry about page breaks. In other words, you can type (or paste) lengthy texts and let the tool break it into separate pages.

Usage

  1. Enter text and give your book a title, author, and type (or "generation
  2. Click the Generate button.
  3. Give yourself a command block in Minecraft.
  4. Click the block to open, and paste the command.
  5. Place an activator next to the block (button, pressure plate, etc.)
  6. Activate command and the book will be placed in your inventory.
